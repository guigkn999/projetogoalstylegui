# GoalStyle ⚽

Projeto acadêmico de e-commerce temático da Copa do Mundo 2026.

## Funcionalidades

- Catálogo de produtos
- Filtro por marcas
- Carrinho de compras
- Modo Dark/Light
- Mapa interativo das lojas
- Atendimento via WhatsApp
- Layout responsivo

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Bootstrap
- Leaflet Maps
